## ADDED Requirements
### Requirement: Распознавание всех форм комментариев
Команда `clean` должна обнаруживать и удалять любую форму комментариев в файлах (`//`, `/* ... */`, HTML `<!-- ... -->` и т.п.), за исключением конструкций, подпадающих под `keepJSDoc` или пользовательскую/системную белую список. Удаление должно проходить без искажения окружающего кода и быть отражено в превьюхах `--dry-run`.

#### Scenario: Удаление многострочных блоков `/* ... */`
- **WHEN** `kodu clean` запускается на TS/JS файле, содержащем блок `/* ... */`, который не помечен как JSDoc или не входит в белый список
- **THEN** блок удаляется, строка, окружённая блоком, остаётся синтаксически корректной, и `--dry-run` показывает тот же диапазон для удаления без применения изменений

#### Scenario: Удаление HTML-комментариев внутри шаблонов
- **WHEN** `kodu clean` запускается на файле шаблонов (HTML, JSX/TSX) с комментарией вида `<!-- ... -->`
- **THEN** HTML-комментарий удаляется, не нарушая структуры шаблона, и последующая генерация/вывод не содержит пустых строк или лишних пробелов
